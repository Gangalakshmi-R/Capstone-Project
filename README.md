# 📊 Capstone Project: Retail Data Analysis & Customer Segmentation

## 📌 Overview
This project consists of two major problem statements focused on retail analytics:

1. **PS1: Walmart Sales Forecasting**
2. **PS2: Customer Segmentation (Online Retail Data)**

The project applies data analysis, machine learning, and visualization techniques to extract insights and support business decision-making.

---

# 🟦 Problem Statement 1: Walmart Sales Forecasting

## 🎯 Objective
- Analyze historical sales data
- Identify key factors influencing sales
- Build a machine learning model to forecast weekly sales
- Predict sales for the next 12 weeks
- Improve inventory management

## ⚙️ Techniques Used
- Feature Engineering (Lag, Rolling Mean)
- Random Forest Regression
- Time-based Train-Test Split
- Forecasting using recursive prediction

## 📂 Outputs
- `processed_data.csv`
- `predictions.csv`
- `future_forecast.csv`
- `sales_model.pkl`
- Power BI Dashboard (`.pbix`)

---

# 🟦 Problem Statement 2: Customer Segmentation

## 🎯 Objective
- Analyze customer purchasing behavior
- Perform RFM analysis
- Segment customers using clustering
- Provide insights for targeted marketing

## ⚙️ Techniques Used
- Data Cleaning & Preprocessing
- RFM Analysis (Recency, Frequency, Monetary)
- Feature Scaling (StandardScaler)
- K-Means Clustering
- Elbow Method for optimal clusters

## 📂 Outputs
- `cleaned_data.csv`
- `rfm_data.csv`
- `customer_segments.csv`
- `kmeans_model.pkl`
- `scaler.pkl`
- Power BI Dashboard (`.pbix`)
